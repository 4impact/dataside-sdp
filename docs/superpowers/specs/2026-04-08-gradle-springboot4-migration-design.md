# Migration: Maven → Gradle (Kotlin DSL) + Spring Boot 4.0.3

**Date:** 2026-04-08  
**Status:** Approved

## Goal

Migrate the `sdp-sink-connector` project from Maven to Gradle (Kotlin DSL) and upgrade Spring Boot from 3.5.5 to 4.0.3 in a single atomic change.

## Approach

Single-shot migration (Approach A): create Gradle files, set Spring Boot 4.0.3, delete `pom.xml` in one commit. The project is small with a limited dependency surface, making simultaneous migration safe.

## Files Changed

| Action | File |
|--------|------|
| Create | `settings.gradle.kts` |
| Create | `build.gradle.kts` |
| Create | `gradle/wrapper/gradle-wrapper.properties` |
| Create | `gradlew` |
| Create | `gradlew.bat` |
| Generate (binary) | `gradle/wrapper/gradle-wrapper.jar` — generated by running `gradle wrapper --gradle-version 8.14` |
| Delete | `pom.xml` |

## Build Configuration

### `settings.gradle.kts`

```kotlin
rootProject.name = "sdp-sink-connector"
```

### `build.gradle.kts`

```kotlin
plugins {
    java
    id("org.springframework.boot") version "4.0.3"
    id("io.spring.dependency-management") version "1.1.7"
}

group = "com.fourimpact"
version = "1.0.0-SNAPSHOT"

java {
    toolchain {
        languageVersion = JavaLanguageVersion.of(25)
    }
}

repositories {
    mavenCentral()
}

dependencies {
    implementation("org.springframework.boot:spring-boot-starter-web")
    implementation("org.springframework.kafka:spring-kafka")
    implementation("org.springframework.retry:spring-retry")
    implementation("org.springframework.boot:spring-boot-starter-aop")
    implementation("com.fasterxml.jackson.core:jackson-databind")
    implementation("com.fasterxml.jackson.datatype:jackson-datatype-jsr310")
    implementation("net.logstash.logback:logstash-logback-encoder:8.1")
    compileOnly("org.projectlombok:lombok")
    annotationProcessor("org.projectlombok:lombok")

    testImplementation("org.springframework.boot:spring-boot-starter-test")
    testImplementation("org.springframework.kafka:spring-kafka-test")
    testImplementation("org.mockito:mockito-core")
    testImplementation("org.wiremock:wiremock-standalone:3.10.0")
    testCompileOnly("org.projectlombok:lombok")
    testAnnotationProcessor("org.projectlombok:lombok")
}

tasks.withType<Test> {
    useJUnitPlatform()
}
```

## Key Design Decisions

- **Lombok**: `compileOnly` + `annotationProcessor` replaces Maven's `<optional>true</optional>`. Also added to `testCompileOnly` + `testAnnotationProcessor` to preserve Lombok in test classes.
- **Pinned versions**: `logstash-logback-encoder:8.1` and `wiremock-standalone:3.10.0` are explicitly versioned as they are not managed by the Spring Boot BOM.
- **Java toolchain**: `JavaLanguageVersion.of(25)` replaces `<java.version>25</java.version>`.
- **Gradle wrapper**: Gradle 8.14, downloaded automatically on first run.
- **No `spring-boot-properties-migrator`**: The application.yml uses standard property names; no known deprecated properties require migration.

## Verification

After files are written, the user runs:

```bash
# Windows
gradlew.bat build
gradlew.bat test

# Unix/WSL
./gradlew build
./gradlew test
```

Note: `JAVA_HOME` must point to a JDK 25 installation. The Gradle toolchain resolver will use it automatically.
